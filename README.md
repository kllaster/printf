# ft_printf
## Description

My implementation of the printf function. <br>

## Usage

```c
int   ft_printf(char const *str, ...)
```

``make`` will compile a ``libftprintf.a`` binary.
<br><br>
ft_printf works with :
* Conversions : ``c`` ``s`` ``p`` ``d`` ``i`` ``u`` ``x`` ``X`` ``%``
* Flags : ``0`` ``*`` ``-``
* Minimum width
* Precision ``.``
